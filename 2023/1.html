Raylib



Raylib is a cross-platform open source libary,the libary was made to create graphical software and games,the source code is written in C(C99), it support many operating system including Windows,Mac os,Linux raylib support more than 50 programming languages.Its development started in 2013 by Ramon Santamaria for a game development course for student with no coding experience,raylib official website is www.raylib.com.
